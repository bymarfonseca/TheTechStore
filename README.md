# TheTechStore
Producto Integrador de Aprendizaje del Laboratorio de Programación Web.

Iniciar pagina en local host
Ingresar a la ruta desde cmd:
cd C:\xampp\htdocs\TheTechStore
Iniciar Node.js con el siguiente comando:
node server.js
Verifica que se conecta correctamente si muestra el siguiente mensaje significa que se hizo la conexion correctamente:
Servidor Node.js corriendo en http://localhost:3000
✅ Conexión exitosa a la base de datos MySQL (Pool iniciado).
